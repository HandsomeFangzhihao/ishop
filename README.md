# PHP原生代码实现的商城

### 比较简陋,学校的作业
- 前台：
  - 注册、登录
  - 用户中心、编辑资料、上传头像
  - 购物车
  - 显示广告轮播、公告文章、商品
  - 搜索
- 后台：
  - 商品增删改查
  - 文章增删改查
  - 广告增删改查
  - 用户增删改查
